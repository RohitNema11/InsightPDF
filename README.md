# InsightPDF

InsightPDF is an AI-powered PDF question-answering app. Upload a PDF, the app splits it into chunks, creates embeddings, and uses retrieval-augmented generation to answer questions with document-backed responses.

## Features

- Upload any PDF file
- Split documents into searchable chunks
- Embed text with HuggingFace embeddings
- Store and query with Chroma
- Use a conversational RAG pipeline for answers

## Tech stack

- Python
- Streamlit
- LangChain
- LangChain Groq
- PyPDFLoader
- HuggingFace embeddings
- Chroma vector store
